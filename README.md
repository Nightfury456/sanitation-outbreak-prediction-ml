🦠 Sanitation-Based Disease Outbreak Prediction

📌 Overview

This project focuses on predicting disease outbreaks using machine learning techniques based on sanitation and public health data. The goal is to identify high-risk regions and enable early preventive measures.

🚀 Features

- Predicts outbreak risk using real-world time-series data
- Uses multiple factors like sanitation, vaccination, testing, and case data
- Implements machine learning models like Random Forest and Gradient Boosting
- Provides risk classification (Low / Medium / High)
- Visualizes results using GIS-based mapping

🛠️ Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- GeoPandas, Folium

📊 Dataset & Approach

- Processed 10,000+ data points across multiple regions
- Created features like lag values and rolling averages
- Trained and evaluated multiple ML models
- Achieved ~90% prediction accuracy
